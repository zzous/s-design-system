# 🧭 버튼 텍스트 사용 가이드

UI에서 버튼 텍스트는 사용자의 행동을 명확하게 전달해야 합니다.
아래는 상황별 버튼 텍스트 기준입니다.

---

## 🎯 1. 저장 관련 버튼

### **저장**
입력 또는 수정된 내용을 시스템에 반영할 때 사용합니다.

- 신규 등록 진행(Submit)
- 수정된 값 저장(Update)
- 작성 완료 후 제출

## 📘 2. 조회 · 확인 화면

### **닫기**
변경 없이 내용만 확인하는 화면에서 사용합니다.

- 상세 조회 모달
- 조회용 팝업
- 정보 확인 모달

## ✏️ 3. 등록 · 수정 화면

### **취소**
입력 또는 수정 중인 작업을 포기하거나 중단할 때 사용합니다.

- 신규 등록 폼 작성 중단
- 수정 화면에서 이탈
- 단계 진행 중 취소


## 📝 4. 알림 · 안내 모달

### **확인**
내용을 읽고 단순히 확인만 하면 되는 경우 사용합니다.

- 경고/주의 메시지
- 단일 선택 알림
- 정보성 안내
