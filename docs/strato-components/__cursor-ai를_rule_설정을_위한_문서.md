## Cursor AI 문서 업데이트 규칙

이 문서는 Cursor AI의 자동 문서화를 위한 규칙을 정의합니다. 컴포넌트 변경 시 다음 사항을 준수해 주세요:

### 문서 업데이트 시점

- 컴포넌트의 Props가 변경될 때
- 컴포넌트의 이벤트가 추가/수정/삭제될 때
- 컴포넌트의 사용 방법이 변경될 때
- 컴포넌트의 주의사항이 추가될 때
- 컴포넌트의 스타일이 크게 변경될 때

### 업데이트 요청 방법

1. 컴포넌트 변경 시 관련 문서 파일을 함께 수정
2. Cursor AI에게 문서 업데이트 요청 시 다음 정보 포함:
   - 변경된 컴포넌트 파일 경로
   - 관련 문서 파일 경로
   - 변경 사항에 대한 설명

### 자동 문서화 범위

Cursor AI는 다음 파일들을 자동으로 관리합니다:
- `docs/strato-components/*.md`: 컴포넌트 마크다운 문서
- `docs/strato-components/*.json`: 컴포넌트 JSON 문서

### 주의사항

- 모든 문서는 이 규칙에 따라 일관성 있게 작성되어야 합니다
- 문서 변경 사항은 반드시 컴포넌트 변경사항과 함께 커밋되어야 합니다
- 문서 업데이트 없이 컴포넌트만 변경하지 않도록 주의해주세요
